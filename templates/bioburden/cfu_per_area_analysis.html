Statistical Definitions
  • Mean: Average CFU value
  • Median: Middle value (50th percentile)
  • Std Dev: Standard deviation (variation from mean)
  • Min/Max: Lowest and highest CFU values
  • Range: Difference between max and min
  • 25th Percentile: 25% of values are below this
  • 75th Percentile: 75% of values are below this
  • Status: Count of normal, alert, and action tests
